inline 방식

 장점: 매우 직관적, 선택하는 과정 없어 속도 빠름

 단점:
 - 선택 과정 없어 중복 기능을 각각 적어야 함
 - 관리가 어려움.
 - 특수 경우 제외, 자주 사용하지 않음.




  Internal 방식1

 장점: 여러 개 선택 후 꾸밀 수 있도록 한번에 효율적으로 작성

 작성법:

<style></style>

 단점: 가독성 떨어짐

  Internal 방식2




   Linking 방식

 장점: 파일 따로 분리 / 가독성이 좋음

 단점: 트래픽 많이 일으킴 => 속도가 느림
